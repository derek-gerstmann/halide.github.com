Detailed Description

Types in the halide type system.

They can be ints, unsigned ints, or floats of various bit-widths (the 'bits' field). They can also be vectors of the same (by setting the 'width' field to something larger than one). Front-end code shouldn't use vector types. Instead vectorize a function.

enum Halide::Type::TypeCode

The basic type code: signed integer, unsigned integer, or floating point.

Note that TypeCode is guaranteed to have values identical to those of halide_type_code_t (HalideRuntime.h), but exists as a separate typedef to preserve source code compatibility.
