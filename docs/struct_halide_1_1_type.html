Detailed Description

Types in the halide type system.

They can be ints, unsigned ints, or floats of various bit-widths (the 'bits' field). They can also be vectors of the same (by setting the 'width' field to something larger than one). Front-end code shouldn't use vector types. Instead vectorize a function.
