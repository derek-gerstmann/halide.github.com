314  /** Add a custom pass to be used during lowering. It is run after
315  * all other lowering passes. Can be used to verify properties of
316  * the lowered Stmt, instrument it with extra code, or otherwise
317  * modify it. The Func takes ownership of the pass, and will call
318  * delete on it when the Func goes out of scope. So don't pass a
319  * stack object, or share pass instances between multiple
