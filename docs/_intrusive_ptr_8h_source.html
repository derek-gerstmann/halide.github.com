46 /** Intrusive shared pointers have a reference count (a
47  * RefCount object) stored in the class itself. This is perhaps more
48  * efficient than storing it externally, but more importantly, it
49  * means it's possible to recover a reference-counted handle from the
50  * raw pointer, and it's impossible to have two different reference
51  * counts attached to the same raw object. Seeing as we pass around
52  * raw pointers to concrete IRNodes and Expr's interchangeably, this
53  * is a useful property.
54  */
