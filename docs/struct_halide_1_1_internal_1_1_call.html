Detailed Description

A function call.

This can represent a call to some extern function (like sin), but it's also our multi-dimensional version of a Load, so it can be a load from an input image, or a call to another halide function. The latter two types of call nodes don't survive all the way down to code generation - the lowering process converts them to Load nodes.
