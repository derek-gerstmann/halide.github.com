Halide::ImageParam::operator Expr ( ) const [inline]

Treating the image parameter as an Expr is equivalent to call it with no arguments.

For example, you can say:

     ImageParam im(UInt(8), 2);
     Func f;
     f = im*2;

This will define f as a two-dimensional function with value at position (x, y) equal to twice the value of the image parameter at the same location.
