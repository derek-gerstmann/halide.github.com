13 /** Some targets treat vectors of bools as integers of the same type
14  * that the boolean operation is being used to operate on. For
15  * example, instead of select(i1x8, u16x8, u16x8), the target would
16  * prefer to see select(u16x8, u16x8, u16x8), where the first argument
17  * is a vector of integers representing a mask. This pass converts
18  * vectors of bools to vectors of integers to meet this
19  * requirement. This is done by injecting intrinsics to convert bools
20  * to architecture-specific masks, and using a select_mask instrinsic
21  * instead of a Select node. Because the masks are architecture
22  * specific, they may not be stored or loaded. On Stores, the masks
23  * are converted to UInt(8) with a value of 0 or 1, which is our
24  * canonical in-memory representation of a bool. */
