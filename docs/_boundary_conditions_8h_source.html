00017 /** namespace to hold functions for imposing boundary conditions on
00018  *  Halide Funcs.
00019  *
00020  *  All functions in this namespace transform a source Func to a
00021  *  result Func where the result produces the values of the source
00022  *  within a given region and a different set of values outside the
00023  *  given region. A region is an N dimensional box specified by
00024  *  mins and extents.
00025  *
00026  *  Three areas are defined:
00027  *      The image is the entire set of values in the region.
00028  *      The edge is the set of pixels in the image but adjacent
00029  *          to coordinates that are not
00030  *      The interior is the image minus the edge (and is undefined
00031  *          if the extent of any region is 1 or less).
00032  *
00033  *  If the source Func has more dimensions than are specified, the extra ones
00034  *  are unmodified.
00035  *
00036  *  Numerous options for specifing the outside area are provided,
00037  *  including replacement with an expression, repeating the edge
00038  *  samples, mirroring over the edge, and repeating or mirroring the
00039  *  entire image.
00040  *
00041  *  TODO: Add support for passing Image<T> and ImageParam, and
00042  *  possibly other types directly to this functions.
00043  */
